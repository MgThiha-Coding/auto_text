# Changelog

## [1.1.0] - 2025-03-16  
### Added  
- Adaptive font scaling (based on both width & height).  
- `maxLines` and `softWrap` support.  
- Improved text overflow handling.  
- Flexible padding with `EdgeInsets`.  

## [1.0.0+1] - 2025-03-16  
- Initial release of the `auto_text` package.
